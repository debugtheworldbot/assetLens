# AssetLens 设计方案头脑风暴

## 项目概述
面向个人用户的资产管理网页应用，设计调性要求：**暖色亲切**，米白底 + 暖橙 + 墨绿 + 沙金 + 雾紫。去金融化、去焦虑感，像本理财日记。

## 方案一：「手账理财」— 纸质手账美学

<text>
**设计运动**：Skeuomorphic Paper / 手账拟物风

**核心原则**：
1. 纸质质感 — 所有卡片和容器模拟手账纸张，带微妙的纸纹纹理和柔和阴影
2. 手绘感装饰 — 使用圆润的手绘风格分隔线、标注箭头、圆圈高亮
3. 温暖的留白 — 大量米白色留白，像翻开一本精心排版的日记本
4. 有机圆角 — 所有元素使用较大的不规则圆角，避免机械感

**色彩哲学**：
- 主背景：米白 `#fbf7f2`，模拟日记本纸张
- 暖橙 `#e57c5a` 作为主要强调色，用于重要数据和CTA
- 墨绿 `#5b9b7d` 用于正面指标（健康、增长）
- 沙金 `#d4a559` 用于警示和中性信息
- 雾紫 `#8a6cae` 用于次要装饰和加密资产
- 卡片背景：纯白 `#ffffff`，带 warm shadow

**布局范式**：
- 左侧固定侧边栏（模拟手账书签标签页），右侧主内容区
- 首页采用不对称网格：左大右小的卡片布局
- 资产列表使用卡片堆叠而非表格，每张卡片像一张便签

**标志性元素**：
1. 侧边栏导航标签页 — 模拟手账的彩色标签贴纸
2. 数据卡片 — 带微妙纸纹底纹和柔和投影的圆角卡片
3. 手绘风格的分隔线和装饰元素

**交互哲学**：
- 页面切换使用翻页般的淡入淡出
- 卡片悬停时微微抬起（translateY + shadow增强）
- 按钮点击有轻微的按压反馈

**动画**：
- 页面进入时卡片从下方依次浮入（stagger animation）
- 数字变化时使用平滑的计数动画
- 饼图从0度展开到完整
- 侧边栏标签切换时有弹性过渡

**字体系统**：
- 标题：Noto Serif SC（衬线体，温暖文雅）
- 数字：DM Mono（等宽，清晰易读的财务数字）
- 正文：Noto Sans SC（无衬线，干净易读）
</text>

## 方案二：「暖光仪表盘」— 现代暖色数据仪表盘

<text>
**设计运动**：Warm Modernism / 暖色现代主义

**核心原则**：
1. 数据优先 — 清晰的信息层级，大数字突出，辅助信息退后
2. 暖色系统 — 用暖色调重新定义金融仪表盘，去冷峻感
3. 呼吸感 — 充足的间距和留白，不堆砌信息
4. 柔和深度 — 使用暖色调阴影和微妙渐变创造层次

**色彩哲学**：
- 背景采用米白到浅米的微渐变 `#fbf7f2` → `#f5efe8`
- 暖橙 `#e57c5a` 作为品牌主色和关键操作色
- 墨绿 `#5b9b7d` 代表安全、健康状态
- 沙金 `#d4a559` 用于警告和注意
- 雾紫 `#8a6cae` 用于加密和特殊类别
- 卡片使用白色 + 暖色调柔和阴影（shadow-warm）

**布局范式**：
- 经典侧边栏 + 内容区布局
- 首页使用 Bento Grid（大小不一的卡片网格），核心数据占大格
- 资产页使用分组列表 + 右侧汇总面板

**标志性元素**：
1. 总资产大数字 — 超大字号 + 暖橙色渐变文字
2. 暖色调的圆环图 — 使用设计文档定义的类别色彩
3. 状态胶囊 — 圆润的状态指示器，带呼吸动画

**交互哲学**：
- 所有交互都有即时的视觉反馈
- 悬停显示详细信息的 tooltip
- 模态框从中心柔和展开

**动画**：
- 数字使用 spring 弹性动画过渡
- 图表数据加载时有流畅的绘制动画
- 卡片进入使用 fade-up + scale 组合
- 状态变化使用 cross-fade 过渡

**字体系统**：
- 标题/大数字：Plus Jakarta Sans（现代几何感，温暖圆润）
- 正文：Inter（可读性极佳）
- 辅助/标签：使用 Plus Jakarta Sans 的轻字重
</text>

## 方案三：「理财花园」— 有机自然美学

<text>
**设计运动**：Organic Naturalism / 有机自然主义

**核心原则**：
1. 自然隐喻 — 资产如花园，需要照料和平衡
2. 柔和曲线 — 大量使用圆形、椭圆、波浪线条
3. 层叠质感 — 半透明叠加、磨砂玻璃效果、柔和光晕
4. 生长感 — 数据展示像植物生长一样自然展开

**色彩哲学**：
- 背景：温暖的奶油白 `#fbf7f2`，像晨光洒在纸上
- 墨绿 `#5b9b7d` 作为主色调（花园的绿），代表健康成长
- 暖橙 `#e57c5a` 作为强调色（花朵的橙），代表活力和行动
- 沙金 `#d4a559` 代表丰收和财富
- 雾紫 `#8a6cae` 代表神秘和未知（加密资产）
- 大量使用这些颜色的半透明变体

**布局范式**：
- 侧边导航使用垂直标签式设计
- 首页使用自由流动的卡片布局，大小错落
- 重要数据使用圆形容器展示（如总资产在一个大圆中）

**标志性元素**：
1. 圆形数据容器 — 总资产等核心数据放在圆形/椭圆形容器中
2. 波浪分隔线 — 区域之间使用柔和的波浪线分隔
3. 半透明叠加卡片 — 卡片使用 backdrop-blur + 半透明背景

**交互哲学**：
- 鼠标移动带动微妙的视差效果
- 元素之间有柔和的连接感
- 操作反馈像水波纹一样扩散

**动画**：
- 页面加载时元素像花朵绽放一样展开
- 数字变化使用流水般的过渡
- 图表使用有机的曲线动画
- 删除操作使用枯萎/收缩动画

**字体系统**：
- 标题：Playfair Display（优雅衬线体）
- 数字：Space Grotesk（现代但有机的几何体）
- 正文：Source Sans 3（清晰自然）
</text>

## 最终选择：方案二「暖光仪表盘」

选择理由：
1. 最贴合设计文档的"去金融化、去焦虑感"要求，同时保持数据仪表盘的专业感
2. Bento Grid 布局在展示多维度资产数据时最高效
3. 暖色现代主义风格在金融应用中独特但不过度花哨
4. 字体选择（Plus Jakarta Sans）既现代又温暖，避免了 Inter 的泛用感
5. 动画方案务实，不会影响性能
